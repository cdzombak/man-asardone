# man asardone

Adds a manual entry for `asardone` to OS X systems.

# install

    git clone git://github.com/cdzombak/man-asardone.git
    cd man-asardone
    chmod +x install.sh
    ./install.sh

`sudo` will prompt for your password as necessary.
